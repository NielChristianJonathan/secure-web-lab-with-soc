# secure-web-lab-with-soc

This project is a learning lab to build a vulnerable web application and monitr security event using a SOC approach.

## Goals
- Understand how web monitoring works
- Learn common web attacks (SQL injection, brute force)
- Monitor and analize logs using Wazuh

## Scope
### 1. Web Application
   - User registration
   - Login system
   - Basic Session management
   - User database
     
### 2. Security Simulation
   - SQL Injection on login
   - Brute-force login attack
     
### 3. SOC Monitoring
   - Logging login activities
   - Monitoring using Wazuh
   - Alert for abnormal activities
  
### 4. Documentation
   - Professional README
   - Architecture Diagram
   - Explanation of attacks and log analysis
